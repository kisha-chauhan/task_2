# task_2
🌸 Data Classification using K-Nearest Neighbors (KNN)
📌 Project Overview

This project demonstrates a basic Machine Learning classification model using the K-Nearest Neighbors (KNN) algorithm. The model is trained on the Iris Flower Dataset to classify flowers into different species based on their sepal and petal measurements.

The project is beginner-friendly and helps understand the complete Machine Learning workflow, including data loading, preprocessing, model training, prediction, evaluation, and visualization.

🎯 Project Objectives
Load and analyze the Iris dataset.
Split the dataset into training and testing sets.
Train a KNN classifier.
Predict flower species.
Evaluate the model using accuracy.
Visualize the classification accuracy using Matplotlib.

🛠️ Technologies Used
Python
Pandas
Scikit-learn
Matplotlib


📈 Machine Learning Algorithm

K-Nearest Neighbors (KNN)

KNN is a supervised Machine Learning algorithm that classifies a new data point by finding the K nearest neighbors and assigning the most common class among them.

In this project:

n_neighbors = 3

means the model looks at the 3 nearest data points before making a prediction.
